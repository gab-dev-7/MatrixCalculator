# Matrix Calculator

A web-based matrix calculator with a React frontend and a Spring Boot backend.

Live application available at: [matrixcalc.gawindlin.com](https://matrixcalc.gawindlin.com)

## Features

- **Matrix Operations**:
  - Addition
  - Subtraction
  - Multiplication
  - Transpose
  - Determinant
  - Inverse
  - Random Matrix Generation

## Tech Stack

- **Frontend**:
  - React
  - Vite

- **Backend**:
  - Java
  - Spring Boot
